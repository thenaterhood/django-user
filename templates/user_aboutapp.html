About the Application

This is a very simple Django application that provides basic user facilities, including registration, user profiles, a login page, and account settings. The UI and functionality is fairly bare, with the hopes that that will make it easier to customize and integrate into other projects.

Features

  • Login page with messages
  • Profile pages with friendly URLs and Gravatar integration
  • Password change functionality
  • Username and email update functionality
  • Registration page with input validation

How to Install

Well, if you're seeing this page then you were probably successful. If by some chance you weren't, look at the README.md file.

How to Use

This application covers the basics of account and profile functionality that most sites provide. It can be extended and reworked to add features and to better fit the needs of the project it will be integrated into.

Data passed to pages:

  • user class instance (passed under the name 'user')
  • form instances where input is accepted, passed under the name "form"
    • The form classes can be found in the forms.py file
    • These forms may be useful to you to use in other apps as well
  • Gravatar profile picture URL passed under the name "gravatar_url"
  • Gravatar profile information, passed under the name "gravatar_profile"
    • You may want to look into Gravatar's API for information on how to use this object.

Provided URLs

You should be able to simply include the user urls.py file in your project's main urls.py file and be good to go. It is expected that you include it so that all /user urls and the /accounts/login page are routed to the user app. You can do otherwise, but your mileage may vary. The listed URLs do not stem from your root (/) URL but from the URL you route to the user app, again preferably /user.

  • /about: this about page
  • /profile/<username>: user profile pages
  • /login: user login page
  • /register: user registration page
  • /welcome: successful login landing page
  • /settings: list of links to different user settings
  • /edit: name and email edit page
  • /password: password change page
  • /logout: logs the user out
